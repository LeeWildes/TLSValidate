TLS 1.0 - Not Approved

Only if needed to support Internet Explorer 7 or 8 which don't support TLSv1.1 and TLSv1.2. Some implementations are susceptible to POODLE.

Microsoft no longer supports either of those versions of Internet Explorer so this really shouldn't ever be needed.

Supported Protocols

  • TLS 1.1
  • TLS 1.2 - Preferred
  • TLS 1.3 - Coming Soon
